## Run Locally

- Clone the Project

```bash
  git clone https://github.com/shljessie/tactilegraphics.git
```

Go to the project directory

1. Create dev.js file in the root directory and add your API key

```javascript
  const api = "YOUR_API_KEY";
```

```bash
  cd tactilegraphics
```

Open `index.html` in your browser
